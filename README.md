# Задание 1.

## Задача 10.

### Текст задачи.

Получить список из произведений цифр соответствующих чисел, хранящихся в
исходном списке. Использовать Seq.map.

### Алгоритм решения.

1. Пользователь вводит строку чисел через пробел..
2. Строка разбивается на массив строк.
3. Массив преобразуется в список.
4. Элементы списка переводятся из строк в числа.
5. Для каждого числа вычисляется произведение его цифр.
6. К каждому элементу списка применяется функция вычисления произведения.
7. Формируется новый список с результатами, затем он же выводится на экран.

# Задание 2.

## Задача 10.

### Текст задачи.

Список содержит строки, обозначающие двоичные числа от 1 до 9. Найти в
десятичной системе счисления сумму элементов списка. Использовать Seq.fold.

### Алгоритм решения.

1. Программа просит пользователя ввести числа.
2. В main введенная строка разбивается на список.
3. Каждое числа в функции binToDec переводится из двоичного в десятичное.
4. Используется функция последовательностей Seq.fold (она проходит по всем элементам, хранит промежуточную сумму и добавляет к ней новые значения).
5. Каждое двоичное число превращается в десятичное, а затем тут же прибавляется к сумме.
6. Seq.fold возвращает эту итоговую сумму, которая в дальнейшем выводится на экран.

# Задание 3.

## Задача 10.

### Текст задачи.

Решить с использованием последовательности задачу на
обработку файлов/каталогов.
Выяснить, есть ли в каталоге и его подкаталогах файл с введенным
названием.

### Алгоритм решения.

1. Программа просит пользователя ввести путь к каталогу и имя искомого файла.
2. В функции Directory.EnumerateFiles получается список всех файлов в каталоге и его подкаталогах, где после формируется последовательность путей к файлам.
3. Для каждого файла извелкается его имя, затем проверяется совпадение имени файла с введенным ползователем именем.
4. Используется функция Seq.fold, которая проверяет, существует ли хотя бы один элемент, удовлетворяющий условию.
5. В случае нахождения файла (true) - поиск останавливается, а на экран выводится "Файл найден.".
